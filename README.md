# Data Structures Library

A Java project implementing fundamental data structures from scratch using generics, without relying on the Java Collections Framework.

This project is intended to strengthen understanding of core data structures, algorithmic behavior, and software engineering practices such as testing and modular project design.

---

## Implemented

**ArrayStack**

An array-based implementation of a stack supporting:

- push
- pop
- peek
- size
- isEmpty
